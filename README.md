# One Time Password
Untuk verifikasi login seperti aplikasi Gojek, Dana, dll.

## Requirement
- PHP 5.6

## Installation
```
$ git clone https://github.com/rzthdev/otpsms
$ cd otpsms
```
## Usage
```
$ Open in browser localhost/otpsms
```
